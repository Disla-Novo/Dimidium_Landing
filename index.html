What is the Dimidium Ecosystem?

Core Compiler Engine

Dimidium features a hardware-agnostic compiler built specifically for additive manufacturing and 3D printing. It parses the custom Bellerophon (.bph) language, allowing you to write a single high-level automation script that compiles cleanly into optimized, target-specific firmware instructions.

Bellerophon IDE

A modular desktop workspace engineered for rapid scripting. Features include real-time syntax validation, live parsing error feedback, customizable theme engines, cross-session file persistence, and built-in hardware boundary profiles.

Extensible Target Architecture

Built from the ground up to decouple your logic from proprietary machine constraints. The compiler targets Klipper (.cfg) and Marlin (.gcode) out of the box, utilizing an extensible modular pipeline designed to adapt to any custom firmware environment.

Bonus modules: Built-in CFG Generator & Gravity Hub (beta).
